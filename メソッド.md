## デフォルト値
```ruby
def say_hello(target = "paiza")
    puts "hello #{target}"
end

say_hello() # 何も値を渡さないと
=> デフォルト値の"paiza"が出力される
```

## シンボルを用いたもの
```ruby
def introduce(name = "村人", msg)
    puts "私は#{name}です。"
    puts "#{msg[:target]}、#{msg[:greeting]}。"
    # p msg
end

introduce("勇者", {target: "世界の皆さん", greeting: "こんにちは"})
=> 私は勇者です。
世界の皆さん、こんにちは。
```
