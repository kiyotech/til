## 基本
```ruby
class Player
    def walk()
        puts "勇者は荒野を歩いていた"
    end
    
    def attack(enemy)
        puts "勇者は#{enemy}を攻撃した"
    end
end
player1 = Player.new()　#インスタンスを実体化
player1.walk()          #メソッドの呼び出し
player1.attack("スライム")
クラス名.メソッド名(引数)
```

---
## initializeメソッド
```ruby
class Player
    def initialize(job) # 初期化メソッド
        @job = job
    end
#　オブジェクトが持つ変数に初期値を設定できる
    def walk()
        puts "#{@job}は、荒野を歩いていた。"
    end
end

player1 = Player.new("戦士")
player1.walk()
```
