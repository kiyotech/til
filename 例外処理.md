## エラーで立ち止まらないようにする

```ruby
puts 1
begin ####
    number = 0
    answer = 100 / number
    puts answer
rescue ZeroDivisionError => e ###
    p e
ensure ###
    puts 2
end
```
```ruby
エラー内容の表示
puts 1
begin
    number = 0
    answer = 100 / number
    puts answer
rescue ZeroDivisionError => e
    puts "0では割り算できません"
    # puts e.message
    # p e.backtrace
    STDERR.puts e.backtrace
ensure
    puts 2
end
```
